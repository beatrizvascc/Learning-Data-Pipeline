# Learning-Data-Pipeline
Experimentos computacionais para observar padrões e comportamentos em séries numéricas.

Este repositório reúne experimentos e estudos pessoais voltados à análise de séries numéricas.
O foco é observar padrões, distribuições e comportamentos emergentes usando ferramentas computacionais.

Projeto em desenvolvimento e aprendizado contínuo.



## Como rodar

1. Instale as dependências:
pip install -r requirements.txt

2. Execute o pipeline principal:
python pipeline_primos.py

3. (Opcional) Interface visual:
streamlit run app_streamlit.py
